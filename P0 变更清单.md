## 一、服务器管理模块

### 1.1 添加/编辑/删除服务器

**新增文件**: `Services/HostManagerService.cs`

**功能实现**:
- ✅ `AddHost()` - 添加新主机，包含完整的参数验证
- ✅ `UpdateHost()` - 编辑现有主机信息
- ✅ `DeleteHost()` - 删除主机（从数据库移除）
- ✅ `GetHostById()` - 根据 ID 获取主机
- ✅ `GetHostsByGroup()` - 按分组获取主机列表
- ✅ `GetOnlineHosts()` - 获取在线主机列表
- ✅ `GetOfflineHosts()` - 获取离线主机列表

**验证规则**:
- 主机名称不能为空
- IP 地址不能为空
- 用户名不能为空
- 密码认证时密码不能为空
- 密钥认证时密钥路径不能为空

### 1.2 SSH 连接测试

**增强文件**: `Services/SshService.cs`

**功能实现**:
- ✅ `TestConnectionAsync()` - 测试 SSH 连接并检测操作系统类型
- ✅ `CheckConnectionAsync()` - 检查现有连接是否有效
- ✅ `ExecuteCommandWithRetryAsync()` - 带重试机制的命令执行
- ✅ 自动检测 Linux 发行版（CentOS/Ubuntu/Debian/RHEL）
- ✅ 友好的错误消息提示

**连接测试流程**:
1. 解密密码/加载密钥
2. 建立 SSH 连接
3. 执行 `uname -a` 检测操作系统
4. 读取 `/etc/os-release` 识别具体发行版
5. 返回检测结果

### 1.3 在线状态检测（心跳机制）

**增强文件**: 
- `Services/SshService.cs` - 心跳定时器
- `Services/HostManagerService.cs` - 心跳检测逻辑
- `Services/DatabaseService.cs` - 心跳时间存储

**功能实现**:
- ✅ `StartHeartbeat()` - 启动心跳检测（默认 5 分钟间隔）
- ✅ `StopHeartbeat()` - 停止心跳检测
- ✅ `HeartbeatCallback()` - 心跳回调函数
- ✅ `UpdateHostHeartbeat()` - 更新主机最后心跳时间
- ✅ `GetOfflineHostIds()` - 获取离线主机列表

**心跳检测逻辑**:
```
每 5 分钟执行一次心跳检测
  ↓
检查所有在线主机的最后心跳时间
  ↓
如果超过 10 分钟未心跳
  ↓
尝试重新连接验证
  ↓
连接失败则标记为离线
```

### 1.4 服务器信息加密存储

## 二、安装流程完善模块

### 2.1 本地包上传逻辑（SFTP）

**增强文件**: `Services/SshService.cs`

**功能实现**:
- ✅ `ConnectAsync()` - 同时建立 SSH 和 SFTP 连接
- ✅ `UploadFileAsync()` - 上传文件到远程服务器
- ✅ 进度回调支持
- ✅ 唯一文件名生成（避免冲突）

**上传流程**:
```
1. 检查本地文件是否存在
2. 计算文件大小
3. 生成唯一文件名
4. 通过 SFTP 上传到 /tmp 目录
5. 实时上报上传进度
```

### 2.2 安装脚本参数传递（环境变量）

**增强文件**: `Services/InstallerService.cs`

**功能实现**:
- ✅ Linux: 使用 `export KEY="value"` 导出环境变量
- ✅ Windows: 使用 `$env:KEY="value"` 设置环境变量
- ✅ 参数自动转义（避免 shell 注入）

**参数传递示例**:
```bash
# Linux
export MYSQL_ROOT_PASSWORD="root123"
export MYSQL_PORT="3306"
bash -c '安装脚本内容'

# Windows PowerShell
$env:MYSQL_ROOT_PASSWORD="root123"; $env:MYSQL_PORT="3306"; 安装脚本内容
```

### 2.3 进度上报解析

**新增文件**: `Services/LogCollector.cs`

**功能实现**:
- ✅ `ProcessOutput()` - 处理 SSH 输出
- ✅ `ParseProgress()` - 解析进度上报
- ✅ `ProgressUpdated` 事件 - 进度更新通知
- ✅ `LogReceived` 事件 - 日志条目通知

**进度上报格式**:
```
PROGRESS:阶段名称：百分比
```

**解析示例**:
```
输入：PROGRESS:安装 MySQL:50
输出：stage="安装 MySQL", percent=50
触发：ProgressUpdated("安装 MySQL", 50)
```

### 2.4 安装结果验证

**增强文件**: `Services/InstallerService.cs`

**功能实现**:
- ✅ `CheckStatusAsync()` - 检查应用状态
- ✅ `IsInstalledAsync()` - 检测是否已安装
- ✅ `IsRunningAsync()` - 检测是否运行中
- ✅ `GetVersionAsync()` - 获取已安装版本

**验证逻辑**（以 Elasticsearch 为例）:
```
1. 检查进程 (pgrep -f elasticsearch)
2. 检查包管理器 (dpkg/rpm)
3. 检查系统服务 (systemctl)
4. 检查可执行文件 (which/test -f)
5. 检查端口占用 (ss -tulnp)
6. 检查 REST API (curl http://127.0.0.1:9200)
7. 检查 Docker 容器
8. 检查配置文件
```

### 3.3 日志持久化

**增强文件**: `Services/DatabaseService.cs`

**功能实现**:
- ✅ `logs` 表创建
- ✅ `SaveTaskLogs()` - 批量保存日志
- ✅ `GetTaskLogs()` - 获取任务日志
- ✅ `GetTaskLogsByLevel()` - 按级别过滤
- ✅ `DeleteTaskLogs()` - 删除任务日志
- ✅ `CleanupOldLogs()` - 清理旧日志

**数据库表结构**:
```sql
CREATE TABLE logs (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    task_id TEXT,
    timestamp DATETIME NOT NULL,
    level INTEGER NOT NULL,
    message TEXT NOT NULL,
    FOREIGN KEY (task_id) REFERENCES tasks(id)
)
```

## 七、API 使用示例

### 7.1 主机管理
```csharp
var hostManager = new HostManagerService(dbService, sshService, logger);

// 添加主机
var host = new RemoteHost
{
    Name = "Test Server",
    IpAddress = "192.168.1.100",
    Port = 22,
    Username = "root",
    EncryptedPassword = EncryptionService.Encrypt("password"),
    AuthType = AuthType.Password,
    OsType = OperatingSystemType.CentOS
};
hostManager.AddHost(host);

// 测试连接
var result = await hostManager.TestConnectionAsync(host);
if (result.Success)
{
    Console.WriteLine("连接成功");
}
```

### 7.2 安装服务
```csharp
var installer = new InstallerService(sshService, logger);

// 执行安装
var task = await installer.InstallAsync(
    host: host,
    app: redisApp,
    parameters: new Dictionary<string, string>
    {
        { "REDIS_PASSWORD", "redis123" },
        { "REDIS_PORT", "6379" }
    },
    localPackagePath: "/path/to/redis.deb"
);

// 监听任务完成
if (task.Status == TaskStatus.Completed)
{
    Console.WriteLine("安装成功");
}
```
