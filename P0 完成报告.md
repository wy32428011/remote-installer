### 3. 日志系统

#### 3.1 日志级别

```csharp
public enum LogLevel
{
    Info,     // 一般信息（蓝色）
    Success,  // 成功信息（绿色）
    Warning,  // 警告信息（黄色）

```

#### 3.2 日志收集器

#### 3.3 日志持久化
