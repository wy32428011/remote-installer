# 远程安装应用需求文档

## 项目概述
- **项目名称**: 远程安装应用
- **技术栈**: C# + WPF
- **创建日期**: 2026-03-13
- **版本**: v1.0

## 核心需求

### 1. 远程连接管理

#### 1.1 多系统支持
- [ ] **Windows Server** - 通过 PowerShell Remoting / WinRM / SSH
- [ ] **CentOS** - 通过 SSH + sudo
- [ ] **Ubuntu** - 通过 SSH + sudo

#### 1.2 连接功能
- [ ] 添加/删除/编辑远程主机
- [ ] 主机信息存储（IP、端口、用户名、密码/密钥、操作系统类型）
- [ ] 连接状态实时检测与显示
- [ ] 批量连接测试
- [ ] 连接超时配置

### 2. 应用安装管理

#### 2.1 支持的应用列表
- [ ] MySQL 8.x
- [ ] Elasticsearch 8.x
- [ ] Redis
- [ ] RabbitMQ
- [ ] Nacos
- [ ] Nginx
- [ ] 支持扩展其他应用

#### 2.2 安装包来源
- [ ] **本地安装包** - 从本地选择安装包文件
- [ ] **远程仓库拉取** - 从配置的仓库地址下载安装包
- [ ] 安装包版本选择

#### 2.3 安装功能
- [ ] 远程上传安装包
- [ ] 自动检测系统环境依赖
- [ ] 执行安装脚本（按操作系统区分）
- [ ] 安装参数配置（端口、路径、账号等）
- [ ] 静默安装支持

### 3. 应用检测与卸载

#### 3.1 安装检测
- [ ] 检测应用是否已安装
- [ ] 检测已安装版本号
- [ ] 检测应用运行状态
- [ ] 检测端口占用情况

#### 3.2 卸载功能
- [ ] 提供一键卸载功能
- [ ] 卸载前确认提示
- [ ] 可选是否保留数据/配置文件
- [ ] 卸载日志记录

### 4. 实时监控

#### 4.1 进度监控
- [ ] 实时显示安装进度百分比
- [ ] 分阶段进度展示（上传中、安装中、配置中、完成）
- [ ] 多任务并发进度展示

#### 4.2 日志查看
- [ ] 实时滚动查看安装日志
- [ ] 日志级别过滤（INFO/WARN/ERROR）
- [ ] 日志下载/保存
- [ ] 日志关键词高亮

### 5. 非功能需求

#### 5.1 安全性
- [ ] 密码/密钥加密存储（AES）
- [ ] 传输加密（SSH/SSL）
- [ ] 操作权限控制
- [ ] 操作审计日志

#### 5.2 可靠性
- [ ] 断线自动重连
- [ ] 任务队列管理
- [ ] 失败自动重试（可配置次数）
- [ ] 超时控制与中断

#### 5.3 用户体验
- [ ] 直观的操作界面
- [ ] 拖拽上传安装包
- [ ] 批量操作支持
- [ ] 快捷键支持
- [ ] 操作撤销/重做

### 6. 技术架构

#### 6.1 客户端（WPF）
- **架构**: MVVM
- **框架**: Prism 或 CommunityToolkit.MVVM
- **异步**: async/await + Task
- **配置存储**: JSON + SQLite

#### 6.2 远程通信
- **Linux (CentOS/Ubuntu)**: SSH - Renci.SshNet 库
- **Windows**: 
  - 方案 1: SSH (OpenSSH Server)
  - 方案 2: PowerShell Remoting (WinRM)
- **文件传输**: SFTP / SCP

#### 6.3 安装脚本管理
- **Linux**: Bash 脚本
- **Windows**: PowerShell 脚本
- 脚本版本管理
- 脚本热更新

#### 6.4 数据处理
- 大文件分片传输
- 进度事件实时上报
- 日志轮转存储

### 7. 界面设计（待细化）

#### 7.1 主窗口
- 远程主机列表（树形/列表）
- 应用市场（可安装应用列表）
- 操作工具栏
- 状态栏（连接数、任务数）

#### 7.2 连接管理窗口
- 新增/编辑连接表单
- 操作系统类型选择
- 测试连接功能
- 连接历史

#### 7.3 系统设置窗口
- 仓库地址配置（URL、用户名、密码）
- 网络代理设置
- 超时时间配置
- 重试次数配置
- 本地缓存目录设置

#### 7.4 任务执行窗口
- 任务列表（进行中/已完成/失败）
- 实时进度条
- 日志查看器（可拆分窗口）
- 任务控制（暂停/继续/取消）
