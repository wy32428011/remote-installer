## 简介

RemoteInstaller 是一款跨平台的远程应用安装管理工具，支持通过 SSH 协议在 Linux 和 Windows 服务器上批量安装、卸载和管理常用中间件应用。

### 支持的应用

| 应用 | Linux | Windows | 版本 |
|------|-------|---------|------|
| MySQL | ✅ | ✅ | 8.0.35 |
| Redis | ✅ | ✅ | 7.2.3 |
| Nginx | ✅ | ✅ | 1.24.0 |
| Elasticsearch | ✅ | ✅ | 8.14.0 |
| RabbitMQ | ✅ | ✅ | 3.12.0 |
| Nacos | ✅ | ✅ | 2.3.0 |

### 主要特性

- 🎨 **Material Design 界面** - 现代化、美观的 WPF 界面
- 🔌 **SSH 连接管理** - 支持密码和密钥认证
- 📦 **应用市场** - 一键安装常用中间件
- 📊 **任务进度** - 实时查看安装进度和日志
- ⚙️ **灵活配置** - 自定义安装参数
- 🌐 **跨平台** - 支持 Linux 和 Windows 目标服务器

## 系统要求

### 客户端（运行 RemoteInstaller）

- **操作系统**: Windows 10/11 或 Windows Server 2016+
- **.NET 版本**: .NET 8.0
- **内存**: 至少 2GB RAM
- **网络**: 能够访问目标服务器的 SSH 端口（默认 22）

### 目标服务器

#### Linux 服务器
- **操作系统**: CentOS 7+, Ubuntu 18.04+
- **SSH 服务**: OpenSSH 7.0+
- **用户权限**: 需要 root 权限或 sudo 权限

#### Windows 服务器
- **操作系统**: Windows Server 2016+
- **SSH 服务**: OpenSSH for Windows
- **PowerShell**: PowerShell 5.1+
- **依赖项**:
  - Elasticsearch: JDK 17+
  - RabbitMQ: Erlang/OTP 25+
  - Nacos: JDK 1.8+

## 快速开始

### 1. 启动程序

双击 `RemoteInstaller.exe` 启动程序。

### 2. 添加主机

1. 点击工具栏 **"➕ 添加主机"**
2. 填写主机信息：
   - **主机名称**: 友好名称（如：生产服务器 -Web01）
   - **操作系统**: 选择 Linux 或 Windows
   - **IP 地址**: 服务器 IP 地址
   - **端口**: SSH 端口（默认 22）
   - **用户名**: 登录用户名（如：root）
   - **密码**: 登录密码
   - **私钥路径**: （可选）SSH 私钥文件路径
3. 点击 **"🔌 测试连接"** 验证连接
4. 点击 **"✅ 添加"** 保存

### 3. 安装应用

1. 在左侧主机列表中选择目标主机
2. 在右侧应用市场找到要安装的应用
3. 点击应用卡片的 **"🔧 配置安装"** 按钮
4. 填写安装参数（如端口、密码等）
5. 点击 **"🚀 开始安装"**

### 4. 查看进度

- **任务列表**: 显示所有进行中的任务
- **进度条**: 实时显示安装进度
- **日志窗口**: 查看详细安装日志

## 使用指南

### 添加主机

#### 密码认证

1. 点击 **"➕ 添加主机"**
2. 填写基本信息
3. 输入密码
4. 测试连接并保存

#### 密钥认证

1. 生成 SSH 密钥对（如果还没有）：
   ```bash
   ssh-keygen -t ed25519 -C "your_email@example.com"
   ```

2. 将公钥复制到目标服务器：
   ```bash
   ssh-copy-id root@target-server
   ```

3. 在 RemoteInstaller 中添加主机：
   - 填写基本信息
   - 在"私钥路径"中选择私钥文件（如：`C:\Users\yourname\.ssh\id_ed25519`）
   - 测试连接并保存

### 安装 MySQL

#### Linux 安装

1. 选择 Linux 主机
2. 点击 MySQL 卡片的 **"🔧 配置安装"**
3. 填写参数：
   - **Root 密码**: 设置 MySQL root 密码
   - **端口**: 默认 3306
4. 点击 **"🚀 开始安装"**

#### Windows 安装

1. 选择 Windows 主机
2. 点击 MySQL 卡片的 **"🔧 配置安装"**
3. 填写参数：
   - **Root 密码**: 设置 MySQL root 密码
   - **端口**: 默认 3306
4. 点击 **"🚀 开始安装"**

### 安装 Elasticsearch

**前置要求**: 目标服务器必须安装 JDK 17+

1. 选择主机
2. 点击 Elasticsearch 卡片的 **"🔧 配置安装"**
3. 填写参数：
   - **HTTP 端口**: 默认 9200
   - **堆内存**: 默认 1g（建议至少 2g）
4. 点击 **"🚀 开始安装"**

### 安装 RabbitMQ

**前置要求**: 目标服务器必须安装 Erlang/OTP 25+

1. 选择主机
2. 点击 RabbitMQ 卡片的 **"🔧 配置安装"**
3. 填写参数：
   - **默认用户**: 默认 guest
   - **默认密码**: 默认 guest
4. 点击 **"🚀 开始安装"**
5. 访问管理界面：`http://服务器 IP:15672`

### 安装 Nacos

**前置要求**: 目标服务器必须安装 JDK 1.8+

1. 选择主机
2. 点击 Nacos 卡片的 **"🔧 配置安装"**
3. 填写参数：
   - **启动模式**: standalone（单机）或 cluster（集群）
4. 点击 **"🚀 开始安装"**
5. 访问 Nacos 控制台：`http://服务器 IP:8848/nacos`
6. 默认账号：`nacos` / `nacos`

### 卸载应用

1. 在应用市场中找到已安装的应用
2. 点击 **"🗑️ 卸载"** 按钮
3. 确认卸载
4. 程序会自动执行卸载脚本并清理相关文件

### 系统设置

点击工具栏 **"⚙️ 系统设置"** 可以配置：

#### 仓库配置
- **仓库地址**: 应用安装包下载地址
- **用户名/密码**: 仓库认证信息

#### 网络设置
- **代理服务器**: HTTP/HTTPS 代理
- **连接超时**: 默认 30 秒
- **重试次数**: 默认 3 次

#### 性能设置
- **最大并发任务数**: 同时执行的任务数（默认 3）
- **本地缓存目录**: 安装包缓存路径

## 常见问题

### Q1: 连接失败 "远程主机强迫关闭了一个现有的连接"

**原因**: SSH 服务器配置问题或网络防火墙

**解决方法**:
1. 检查目标服务器 SSH 服务是否运行：
   ```bash
   systemctl status sshd  # Linux
   ```

2. 检查防火墙设置：
   ```bash
   firewall-cmd --list-ports  # CentOS
   ufw status  # Ubuntu
   ```

3. 确认 SSH 允许密码登录（/etc/ssh/sshd_config）：
   ```
   PasswordAuthentication yes
   ```

### Q2: 权限不足 "Permission denied"

**原因**: 用户没有足够的权限

**解决方法**:
1. 使用 root 用户或具有 sudo 权限的用户
2. 检查 SSH 密钥权限：
   ```bash
   chmod 600 ~/.ssh/id_rsa
   chmod 700 ~/.ssh
   ```

### Q3: Elasticsearch 启动失败

**原因**: 内存不足或 Java 版本不兼容

**解决方法**:
1. 检查 Java 版本：
   ```bash
   java -version
   ```

2. 增加堆内存（配置时设置更大的值）

3. 检查系统内存：
   ```bash
   free -h
   ```

### Q4: RabbitMQ 安装失败

**原因**: Erlang 版本不兼容

**解决方法**:
1. 检查 Erlang 版本（需要 25+）：
   ```bash
   erl -version
   ```

2. 下载并安装 Erlang/OTP 25+：
   https://www.erlang.org/downloads

### Q5: 文件上传失败

**原因**: SFTP 服务问题

**解决方法**:
- 当前版本使用 Base64 编码传输文件，不依赖 SFTP
- 如果仍然失败，检查 SSH 连接状态

### Q6: 如何批量安装？

**当前版本**: 支持单台服务器安装

**未来计划**: 将支持选择多台服务器批量安装

### Q7: 安装日志在哪里？

**位置**: 
- 界面底部日志窗口（实时查看）
- 文件路径：`%APPDATA%\RemoteInstaller\logs\`

### Q8: 如何更新应用版本？

**方法**:
1. 先卸载旧版本
2. 重新安装新版本

**未来计划**: 将支持直接升级
